# Codex Assistant CLI 🚀

Automatiza tu flujo de desarrollo con **OpenAI Codex 2025**: generación de código, refactorización, pruebas y ejecución segura en un entorno aislado.

---

## ⚡ Descripción

Este proyecto ofrece una **CLI** y utilidades en Python para:

* Generar y modificar código de forma interactiva.  
* Ejecutar y depurar scripts en un *sandbox* protegido.  
* Analizar, documentar y probar tu base de código automáticamente.

---

## 📝 Requisitos

| Componente          | Versión mínima |
| ------------------- | -------------: |
| Python              | **3.10**       |
| Cuenta de OpenAI    | Acceso a **Codex 2025** |
| Variable de entorno | `OPENAI_API_KEY` |

## 🔧 Instalación rápida

```bash
# 1 · Clona el repositorio
git clone https://github.com/tu-usuario/codex-assistant-cli.git
cd codex-assistant-cli

# 2 · Activa un entorno virtual (opcional)
python -m venv .venv
# Linux/macOS
source .venv/bin/activate
# Windows
.venv\Scripts\activate

# 3 · Instala las dependencias
pip install -r requirements.txt
# ChatGPT
